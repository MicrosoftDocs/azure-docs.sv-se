Utför följande steg i Azure Portal för att visa en enhets händelse. 

1. I Azure Portal går du till Azure Stack Edge/Data Box Gateway-resursen och går sedan till **övervakning > enhets händelser**.
2. Välj en händelse och visa aviseringsinformationen. Åtgärda varningsvillkoret med lämpliga åtgärder.
