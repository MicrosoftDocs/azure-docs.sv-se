Med anpassade avbildningar i Azure DevTest Labs kan du snabbt skapa virtuella datorer utan att vänta tills all nödvändig program vara är installerad på mål datorn. Med anpassade avbildningar kan du förinstallera all program vara som du behöver i en VHD-fil och sedan använda VHD-filen för att skapa en virtuell dator. Eftersom program varan redan är installerad är tiden för att skapa virtuella datorer mycket snabbare. Dessutom kan anpassade avbildningar användas för att klona virtuella datorer genom att skapa en anpassad avbildning från en virtuell dator och sedan skapa virtuella datorer baserat på den anpassade avbildningen.
