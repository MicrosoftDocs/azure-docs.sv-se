> Om du vill undvika avgifter när inga jobb körs **anger du de lägsta noderna till 0**. Med den här inställningen kan Azure Machine Learning ta de noderna när de inte används. Alla värden som är större än 0 behåller det antalet noder som körs, även om de inte används.
