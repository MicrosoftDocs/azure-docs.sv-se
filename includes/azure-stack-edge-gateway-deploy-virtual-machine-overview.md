Du kan skapa och hantera virtuella datorer på en Azure Stack Edge-enhet med hjälp av API:er. Dessa API: er är standard Azure Resource Manager-API: er som anropas med den lokala Azure Stack gräns slut punkten. Azure Resource Manager-API:erna ger ett enhetligt hanteringslager som i det här fallet gör att du kan skapa, uppdatera och ta bort virtuella datorer i en lokal prenumeration som finns på enheten. Du kan ansluta till den Azure Resource Manager som körs i Azure Stack Edge via Azure PowerShell-cmdletar.
