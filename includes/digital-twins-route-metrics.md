## <a name="monitor-event-routes"></a>Övervaka händelse vägar

Routnings mått som antal, svars tid och frekvens kan visas i [Azure Portal](https://portal.azure.com/). 

Från portalens start sida söker du efter din Azure Digital-instansen för att hämta information. Välj alternativet **mått** från den Azure Digital-instansen navigerings meny till vänster för att öppna sidan *mått* .

:::image type="content" source="../articles/digital-twins/media/troubleshoot-metrics/azure-digital-twins-metrics.png" alt-text="Skärm bild som visar mått sidan för Azure Digital-enheter.":::

Härifrån kan du visa måtten för din instans och skapa anpassade vyer.

Mer information om hur du visar Azure Digitals dubbla mått finns i [*instruktion: Visa mått med Azure Monitor*](../articles/digital-twins/troubleshoot-metrics.md).
