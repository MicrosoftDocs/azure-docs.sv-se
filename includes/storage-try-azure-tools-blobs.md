> [!TIP]
> 
> Hantera Azure Blob Storage-resurser med Azure Storage Explorer. 
> [Azure Storage Explorer](https://azure.microsoft.com/features/storage-explorer/) är en kostnads fri, fristående app från Microsoft som gör det möjligt att [Hantera Azure Blob Storage-resurser](../articles/vs-azure-tools-storage-explorer-blobs.md). Med hjälp av Azure Storage Explorer kan du visuellt skapa, läsa, uppdatera och ta bort BLOB-behållare och blobbar, samt hantera åtkomst till dina BLOB-behållare och blobbar.
