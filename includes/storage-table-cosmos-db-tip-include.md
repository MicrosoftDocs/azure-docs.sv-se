> [!TIP]
> Artikelns innehåll gäller för den ursprungliga Azure Table Storage-tjänsten. Det finns dock ett alternativt erbjudande för Table Storage: Azure Cosmos DB Tabell-API. Detta API ger högre prestanda och tillgänglighet, global distribution och automatiska sekundära index. Den är också tillgänglig i ett konsumtions fritt [Server](../articles/cosmos-db/serverless.md) läge. Det finns vissa [funktions skillnader](../articles/cosmos-db/table-api-faq.md#table-api-vs-table-storage) mellan Tabell-API i Azure Cosmos DB och Azure Table Storage. Mer information finns i [Azure Cosmos DB tabell-API](../articles/cosmos-db/table-introduction.md).
