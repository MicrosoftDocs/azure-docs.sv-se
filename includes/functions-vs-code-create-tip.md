> [!TIP]
> Som standard skapas de Azure-resurser som krävs av din Function-app baserat på namnet på den Function-app som du anger. Som standard skapas de också i samma nya resurs grupp med Function-appen. Om du antingen vill anpassa namnen på dessa resurser eller återanvända befintliga resurser måste du i stället [publicera projektet med avancerade alternativ för att skapa](../articles/azure-functions/functions-develop-vs-code.md#enable-publishing-with-advanced-create-options).
