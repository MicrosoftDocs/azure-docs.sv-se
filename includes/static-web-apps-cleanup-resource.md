Om du inte kommer att fortsätta att använda det här programmet kan du ta bort Azures statiska webb program resurs genom följande steg:

1. Öppna [Azure-portalen](https://portal.azure.com)
1. Sök efter ditt program med det namn du angav tidigare i det övre Sök fältet
1. Klicka på appen
1. Klicka på knappen **ta bort**
1. Bekräfta borttagnings åtgärden genom att klicka på **Ja**
