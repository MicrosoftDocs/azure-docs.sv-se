Azure Cloud Shell är ett webbläsarbaserat gränssnitt som gör det möjligt att hantera och utveckla Azure-resurser.

Cloud Shell erbjuder en webbläsaraktiverade, förkonfigurerad gränssnitts upplevelse som kan användas för att hantera Azure-resurser utan att behöva installera, konfigurera och underhålla en dator själv.

Cloud Shell etablerar datorer per begäran och att dator status inte bevaras mellan sessioner. Eftersom Cloud Shell skapats för interaktiva sessioner avslutas gränssnitten automatiskt efter 20 minuters inaktivitet i gränssnittet.
