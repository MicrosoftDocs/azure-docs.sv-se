# <a name="backup-solutions-for-azure-vmware-solution-virtual-machines-vms"></a>Säkerhetskopieringslösningar för Azure VMware Solution virtuella datorer (VM)

En viktig princip för Azure VMware Solution är att du kan fortsätta att använda dina investeringar och dina favorit-VMware-lösningar som körs i Azure. Oberoende programvaruleverantör (ISV)-teknikstöd, verifierat med Azure VMware Solution, är en viktig del av den här strategin. 

Våra säkerhetskopieringspartner har branschledande lösningar för säkerhetskopiering och återställning i VMware-baserade miljöer. Kunderna har brett infört dessa lösningar för sina lokala distributioner. Nu har dessa partner utökat sina lösningar till Azure VMware Solution och använt Azure för att tillhandahålla en lagringsplats för säkerhetskopiering och ett lagringsmål för långsiktig kvarhållning och arkivering.

Nätverkstrafik för säkerhetskopiering Azure VMware Solution virtuella datorer och säkerhetskopieringslagringsplatsen i Azure färdas över en länk med hög bandbredd och låg latens. Replikeringstrafik mellan regioner färdas över det interna Azure-backplansnätverket, vilket sänker bandbreddskostnaderna för användarna.
