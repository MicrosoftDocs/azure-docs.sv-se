# <a name="enabling-synapse-workspace-features-on-a-dedicated-sql-pool-formerly-sql-dw-in-your-region"></a>Aktivera Synapse-arbetsytefunktioner i en dedikerad SQL-pool (tidigare SQL DW) i din region.

## <a name="regions-not-supported"></a>Regioner som inte stöds 
I följande tabell sammanfattas efter region där möjligheten att aktivera Synapse-arbetsytans funktioner på en befintlig dedikerad SQL-pool (tidigare SQL DW) inte stöds. Kom tillbaka för att se när din region blir tillgänglig.

\* anger att ett specifikt schema för regionen för närvarande inte är tillgängligt.

| Region | Tillgänglighet för Synapse-arbetsyta |
|:--- |:--- |:--- |
| Australien, centrala 1 |\* |
| Australien, centrala 2 |\* |
| Kina, östra 2 |\* |
| Kina, norra 2 |\* |
| Södra Indien |\*|
| Indien, västra |\*|
| Sydkorea, södra |\* |
| Schweiz, västra |\* |
| Förenade Arabemiraten, norra |\*|
 
 
> [!NOTE]
> Regionlistan ovan gäller endast för regioner där aktivering av Synapse-arbetsytefunktioner inte är tillgänglig för närvarande.
