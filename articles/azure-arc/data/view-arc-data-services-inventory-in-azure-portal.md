# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Visa inventering av dina instanser i Azure Portal

När du har laddat upp dina [mått, loggar](upload-metrics-and-logs-to-azure-monitor.md)eller [användning](view-billing-data-in-azure.md)kan du Visa din instans som en Azure-resurs från Azure Portal. Om du vill visa din resurs i Azure Portal går du till den särskilda URL [https://portal.azure.com](https://portal.azure.com) : en och utför följande steg:

1. Gå till **alla tjänster**.
1. Sök efter typ av databas instans.
1. Lägg till typen i dina favoriter.
1. På den vänstra menyn väljer du typ av instans.
1. Visa dina instanser i samma vy som dina andra Azure SQL-eller Azure PostgreSQL-storskaliga resurser (Använd filter för en detaljerad vy).
