# <a name="view-azure-arc-data-controller-resource-in-azure-portal"></a>Visa resursen Azure Arc data Controller i Azure Portal

När du har gjort dina första [mått eller loggar överför till Azure](upload-metrics-and-logs-to-azure-monitor.md) eller [Uppladdad användning](view-billing-data-in-azure.md)kan du se att den Azure Arc data Controller-resurs som har skapats i Azure i Azure Portal.

Öppna Azure Portal med hjälp av den särskilda URL: en:  [https://portal.azure.com](https://portal.azure.com) .

Sök efter din data enhet efter namn i Sök fältet och klicka på den.
