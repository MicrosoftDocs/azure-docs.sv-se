# <a name="get-it-now---add-an-app-from-the-azure-marketplace"></a>Hämta nu – lägg till en app från Azure Marketplace

Du är nästan klar! 

Om du försöker använda Azure AD som identitetsprovider för en app är du på rätt plats. Du behöver bara lägga till den i din Azure AD-klientorganisation. Om du vill lära dig hur du gör detta följer du snabbstartsserien här: [Visa appar i din Azure AD-klientorganisation.](view-applications-portal.md)

Om du försöker installera en app på din lokala dator eller mobila enhet är du på fel plats. Den Azure Marketplace är utformad för organisationer som använder appar med sina Azure-klientorganisationer. För persondatorer och enheter kan du kolla Microsoft Apps [store](https://www.microsoft.com/store/apps).
