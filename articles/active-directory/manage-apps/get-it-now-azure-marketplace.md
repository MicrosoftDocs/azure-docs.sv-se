# <a name="get-it-now---add-an-app-from-the-azure-marketplace"></a>Hämta nu – Lägg till en app från Azure Marketplace

Nästan klart! 

Om du försöker använda Azure AD som identitets leverantör för en app är du på rätt plats. Du behöver bara lägga till den till din Azure AD-klient. Läs om hur du gör detta genom att följa snabb starts serien här: [Visa appar i din Azure AD-klient](view-applications-portal.md).

Om du försöker installera en app på din lokala dator eller mobila enhet är du på fel plats. Azure Marketplace är utformat för organisationer som använder appar med sina Azure-klienter. För persondatorer och enheter kan du kolla in [Microsoft Apps-butiken](https://www.microsoft.com/store/apps).
