# <a name="customer-data-storage-for-australian-and-new-zealand-customers-in-azure-active-directory"></a>Kunddatalagring för kunder i Australien och Nya Zeeland i Azure Active Directory 

Azure Active Directory (Azure AD) lagrar sina kunddata på en geografisk plats baserat på det land som du angav när du registrerade dig för en Microsoft Online-tjänst. Microsoft Online-tjänster omfattar Microsoft 365 och Azure. 

Information om var Azure AD och Microsoft-tjänster data finns finns i avsnittet Var finns dina [data?](https://www.microsoft.com/trustcenter/privacy/where-your-data-is-located) i Microsoft Trust Center.

Från den 26 februari 2020 började Microsoft lagra Azure AD:s kunddata för nya klienter med en faktureringsadress för Australien eller Nya Zeeland i de australiska datacentren.

Dessutom stöder vissa Azure AD-funktioner ännu inte lagring av kunddata i Australien. Gå till [Azure AD-datakartan](https://msit.powerbi.com/view?r=eyJrIjoiYzEyZTc5OTgtNTdlZS00ZTVkLWExN2ItOTM0OWU4NjljOGVjIiwidCI6IjcyZjk4OGJmLTg2ZjEtNDFhZi05MWFiLTJkN2NkMDExZGI0NyIsImMiOjV9)för specifik funktionsinformation. Till exempel Microsoft Azure AD Multi-Factor Authentication kunddata i USA och bearbetar dem globalt. Se [Datahemhemlighet och kunddata för Azure AD Multi-Factor Authentication.](../authentication/concept-mfa-data-residency.md)

> [!NOTE]
> Microsofts produkter, tjänster och tredjepartsprogram som integreras med Azure AD har åtkomst till kunddata. Utvärdera varje produkt, tjänst och program som du använder för att avgöra hur kunddata bearbetas av den specifika produkten, tjänsten och programmet och om de uppfyller företagets krav på datalagring. Läs mer om Microsoft-tjänsternas datahemvist i avsnittet [Var finns dina data?](https://www.microsoft.com/trustcenter/privacy/where-your-data-is-located) på Microsoft Trust Center.
