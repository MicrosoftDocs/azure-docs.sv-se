# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

I den här artikeln beskrivs orsaker och lösningar för **503003 PartitionNotFound** -fel.

## <a name="symptoms"></a>Symtom

Begär anden till IoT Hub fungerar inte med fel **503003-PartitionNotFound**.

## <a name="cause"></a>Orsak

Det här felet är internt för att IoT Hub och är sannolikt tillfälligt. Se [IoT Hub interna Server fel orsaken](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause).

## <a name="solution"></a>Lösning

Se [lösningar för att IoT Hub interna Server fel](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution).
