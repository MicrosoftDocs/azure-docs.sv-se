I den här artikeln beskrivs Microsoft partner företag som är integrerade med Azure Data Lake Storage. Dessa partner lösningar avser arbets belastningar som moderna informations lager arbets belastningar, avancerad analys och analys i real tid. Dessa partners utnyttjar det [hierarkiska namn området](../../../blobs/data-lake-storage-namespace.md) i Azure Storage för att optimera sin lösning och köra den effektivt i Azure.
