# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Azure Storage partner för primär och sekundär lagring

I den här artikeln beskrivs Microsoft partner företag som levererar en NAS-lösning (Network Attached Storage) eller storage area network (SAN). Lösningen kan vara lokal, i Azure eller en hybrid lösning som använder Azure Storage som en kostnads effektiv nivå. Dessa lösningar kan göra det möjligt för kunder att använda samma lösning i vilken miljö som helst.
