# <a name="azure-policy-regulatory-compliance-controls-for-azure-kubernetes-service-aks"></a>Azure Policy för regelefterlevnad för Azure Kubernetes Service (AKS)

[Regelefterlevnad i Azure Policy](../governance/policy/concepts/regulatory-compliance.md) tillhandahåller initiativdefinitioner (inbyggda) som skapas och hanteras av Microsoft för *efterlevnadsdomäner* och säkerhetskontroller som är relaterade till olika efterlevnadsstandarder. På den här sidan visas Azure Kubernetes Service (AKS) efterlevnadsdomäner och säkerhetskontroller.

Du kan tilldela de inbyggda för en säkerhetskontroll **individuellt för** att göra dina Azure-resurser kompatibla med den specifika standarden.
