# <a name="texconv---texture-conversion-tool"></a>Konverterings verktyg för TexConv-textur

Verktyget TexConv är ett externt kommando rads verktyg för att utföra satsvis struktur konverteringar.
Verktyget finns i den här git-lagringsplatsen: [GitHub-lagringsplats för TexConv-verktyget](https://github.com/microsoft/DirectXTex/wiki/Texconv).

För fjärrrendering behöver texturer inte konverteras eftersom konverterings steget kan hantera vanliga bild fil format och konvertera texturer till lämpligt körnings format. Det här verktyget kan dock vara till hjälp för att konvertera texturer lokalt med anpassade inställningar.
