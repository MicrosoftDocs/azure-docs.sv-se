# <a name="azure-policy-regulatory-compliance-controls-for-azure-image-builder"></a>Azure Policy regler för regelefterlevnad för Azure Image Builder

[Regelefterlevnad i Azure Policy](../governance/policy/concepts/regulatory-compliance.md) tillhandahåller Microsoft skapade och hanterade initiativdefinitioner,  så kallade  inbyggda , för _efterlevnadsdomäner_ och säkerhetskontroller som är relaterade till olika efterlevnadsstandarder. På den här sidan visas **efterlevnadsdomäner** **och säkerhetskontroller** för Azure Image Builder. Du kan tilldela de inbyggda för en säkerhetskontroll **individuellt för** att göra dina Azure-resurser kompatibla med den specifika standarden.
