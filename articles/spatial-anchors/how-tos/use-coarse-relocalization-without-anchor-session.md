# <a name="use-coarse-relocalization-without-an-anchor-session"></a>Använd grov omlokalisering utan en Anchor-session

Objektet sensor-finger avtryck provider som är en del av den [grova Omlokaliserings](../concepts/coarse-reloc.md) funktionen kan användas utan att kopplas till en Azure-session med spatiala ankare.
Följande kodfragment visar hur sensor leverantören kan nås för att få en uppskattning av enhetens position:

# <a name="c"></a>[C#](#tab/csharp)

```csharp
// Game about to start, start tracking the sensors
sensorProvider.Start();

// Game loop
while (m_isRunning)
{
    // Get the GPS estimate
    GeoLocation geoPose = sensorProvider.GetLocationEstimate();

    // Paint it on the map
    drawCircle(
        x: geoPose.Longitude,
        y: geoPose.Latitude,
        radius: geoPose.HorizontalError);
}

// Game ended, no need to track the sensors anymore
sensorProvider.Stop();
```
