# <a name="what-is-azure-netapp-files"></a>Vad är Azure NetApp Files

Azure NetApp Files-tjänsten är en företagsanpassad, avgiftsbelagd fillagringstjänst med höga prestanda. Azure NetApp Files fungerar med alla typer av arbetsbelastning och är högtillgänglig. Du kan välja service- och prestandanivåer och ställa in ögonblicksbilder via tjänsten. 

## <a name="learn-how-to-use-azure-netapp-files"></a>Lär dig hur du använder Azure NetApp Files

I dokumentationen kring Azure NetApp Files finns instruktioner kring hur du skapar och hanterar volymer med hjälp av Azure NetApp Files. 

> [!IMPORTANT] 
> Alla uppgifter som beskrivs i den här dokumentationen kräver att du har beviljats åtkomst till tjänsten Azure NetApp Files. Om du vill begära åtkomst till tjänsten läser du [skicka en Waitlist-begäran om åtkomst till tjänsten](azure-netapp-files-register.md#waitlist).
