# <a name="create-a-media-services-and-storage-account-with-a-private-link"></a>Skapa ett Media Services- och lagringskonto med en Private Link

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

Skapa ett Media Services-konto och lagringskonto med privata länkar till ett VNet. Arm Azure Resource Manager mallen (Arm) uppsättningar även DNS för båda privata länkarna. Slutligen skapar mallen en virtuell dator så att användaren kan prova de privata länkarna.

## <a name="prerequisites"></a>Förutsättningar

Läs [snabbstart: Skapa och distribuera ARM-mallar med hjälp av Azure Portal](https://docs.microsoft.com/azure/azure-resource-manager/templates/quickstart-create-templates-use-the-portal).

## <a name="limitations"></a>Begränsningar

- Mallen Media Services till exempel bara Private Link nyckelleverans.
- Ingen nätverkssäkerhetsgrupp skapas för den virtuella datorn.
- Nätverksåtkomstkontroll har inte konfigurerats för lagringskontot eller nyckelleveransen.

Mallen skapar:

- Ett Media Services och ett lagringskonto (som vanligt)
- Ett VNet med ett undernät
- För både Media Services och lagringskontot:
  - Privata slutpunkter
  - Privat DNS zoner
  - Länkar mellan länkar (för att ansluta privata DNS-zoner till det virtuella nätverket)
  - Privat DNS zongrupper (för att utlösa automatisk generering av DNS-poster i privata DNS-zoner)
- En virtuell dator (med associerad offentlig IP-adress och nätverksgränssnitt)
