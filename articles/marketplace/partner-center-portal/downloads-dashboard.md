# <a name="downloads-dashboard-in-commercial-marketplace-analytics"></a>Hämta instrument panel i kommersiell Marketplace-analys

Den här artikeln innehåller information om instrument panelen för nedladdningar i Partner Center. Den här instrument panelen visar en lista över dina nedladdnings begär Anden under de senaste 30 dagarna.

Öppna instrument panelen för hämtningar genom att öppna **[analys](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)** instrument panelen under den kommersiella marknads platsen.

>[!NOTE]
> Detaljerade definitioner av analys terminologi finns i [vanliga frågor och terminologi för affärs platsers analys](../analytics-faq.md).

## <a name="downloads-dashboard"></a>Ladda ned instrumentpanel

Instrument panelen för **nedladdningar** i **analys** menyn visar begär Anden om hämtningar som innehåller över 1000 rader med kund-eller order data.

Du får ett popup-meddelande som innehåller en länk till instrument panelen för **nedladdningar** när du begär en nedladdning med över 1000 rader med data. Dessa data hämtningar är tillgängliga under en 30-dagarsperiod och tas sedan bort.

## <a name="lifetime-export-of-commercial-marketplace-analytics-reports"></a>Livs längd export av affärs marknads analys rapporter

På sidan nedladdningar kan slutanvändaren göra följande:

- Livs längd export av affärs marknads analys rapporter i CSV-och TSV-format.
- Export av affärs marknads analys rapporter för alla datum intervall.
- Export av affärs marknads analys rapporter för 6-eller 12-månaders varaktighet.

Stöd för livs längds export funktioner för analys rapporter:

| Rapport | Export livs längd | Varaktighet baserat på datum |
| - | - | - |
| Order | ![Grön bock markering](media/check-green-yes.png) | ![Grön bock markering](media/check-green-yes.png) |
| Kunder | ![Grön bock markering](media/check-green-yes.png) | ![Grön bock markering](media/check-green-yes.png) |
| Marknads insikter | ![Grön bock markering](media/check-green-yes.png) | ![Grön bock markering](media/check-green-yes.png) |
| Användning | ![Svart X-tecken](media/check-black-no.png) | Högst ett år |
