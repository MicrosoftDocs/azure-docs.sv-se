# <a name="migrate-to-azure-app-service"></a>Migrera till Azure App Service

Med hjälp av [App Service Migration Assistant](https://azure.microsoft.com/services/app-service/migration-assistant/) kan du migrera din lokala App till Azure App Service. App Service Migration Assistant är utformad för att förenkla din resa till molnet via en kostnads fri, enkel och snabb lösning för att migrera program från en lokal plats till molnet.

Med Azure App Service Migration Assistant kan du snabbt:

- Sök igenom appens URL för att bedöma om det är en bra kandidat för migrering
- Hämta Migration Assistant för att starta migreringen.
- Använd verktyget för att köra beredskaps kontroller och allmän utvärdering av appens konfigurations inställningar
- Migrera din app eller webbplats till Azure App Service via verktyget.

[Se hur du migrerar webbappar till Azure App-tjänsten](https://www.youtube.com/watch?v=9LBUmkUhmXU).

Nästa steg:  [Migrera ett lokalt webb program till Azure App Service](https://docs.microsoft.com/learn/modules/migrate-app-service-migration-assistant/)
