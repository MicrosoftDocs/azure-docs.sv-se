# <a name="azure-policy-regulatory-compliance-controls-for-azure-app-service"></a>Azure Policy för regelefterlevnad för Azure App Service

[Regelefterlevnad i Azure Policy](../governance/policy/concepts/regulatory-compliance.md) tillhandahåller Microsofts skapade och hanterade initiativdefinitioner,  så  kallade inbyggda , för _efterlevnadsdomäner_ och säkerhetskontroller relaterade till olika efterlevnadsstandarder. På den här sidan visas **efterlevnadsdomäner** **och säkerhetskontroller** för Azure App Service. Du kan tilldela de inbyggda för en säkerhetskontroll **individuellt för** att göra dina Azure-resurser kompatibla med den specifika standarden.
